# Cocos Creator 3.8 Skills

| [cocos-asset-skill](./cocos-asset-skill/) | 资源系统与资源类型 | 核心 |

## 学习路径

### 入门路径
1. **cocos-intro-skill** - 引擎介绍与入门
2. **cocos-scene-skill** - 场景制作基础
3. **cocos-asset-skill** - 资源系统
4. **cocos-script-skill** - 脚本编写

### 核心开发路径
5. **cocos-ui-skill** - UI 系统
6. **cocos-animation-skill** - 动画系统
7. **cocos-physics-skill** - 物理系统
8. **cocos-publish-skill** - 发布部署

### 高级特性路径
9. **cocos-render-skill** - 图形渲染
10. **cocos-2d-render-skill** - 2D 渲染
11. **cocos-audio-skill** - 音频系统
12. **cocos-particle-skill** - 粒子特效
13. **cocos-tween-skill** - 缓动动画
14. **cocos-asset-manager-skill** - 资源管理
15. **cocos-terrain-skill** - 地形系统

### 专家路径
16. **cocos-i18n-skill** - 本地化
17. **cocos-editor-extension-skill** - 编辑器扩展
18. **cocos-advanced-skill** - 进阶主题
