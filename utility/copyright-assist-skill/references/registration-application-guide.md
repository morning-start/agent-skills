# 登记申请指南

> 来源：https://blog.csdn.net/aigccleaner/article/details/150586321

## 入口

登录后进入：版权登记 → 软件登记 → R11【计算机软件著作权登记申请】

## 第1页：选择办理身份

根据账号类型选择对应的办理身份。

## 第2页：基本信息

### 字段说明

| 字段 | 选项 | 说明 |
|------|------|------|
| 取得方式 | 原始取得 / 继受取得 | 默认【原始取得】 |
| 权利范围 | 全部权利 / 部分权利 | 默认【全部权利】 |
| 软件全称 | — | 建议以"软件"、"系统"或"平台"结尾 |
| 软件简称 | — | 可选 |
| 版本号 | — | 如 V1.0 |

**填写错误会导致补正，务必仔细。**

- **原始取得**：自己开发的软件
- **继受取得**：通过受让、继承或承受方式取得著作权

## 第3页：软件开发信息

### 字段说明

| 字段 | 选项 | 个人独立申请推荐 |
|------|------|-----------------|
| 软件分类 | 应用软件 / 操作系统 / 嵌入式软件 / 中间件 | 应用软件 |
| 软件说明 | 原创 / 修改（含翻译软件、合成软件） | 原创 |
| 开发方式 | 独立开发 / 合作开发 / 委托开发 / 下达任务开发 | 独立开发 |
| 完成日期 | — | 应在提交日期之前几日 |
| 发表状态 | 未发表 / 已发表 | 未发表 |

### 软件分类说明

| 分类 | 说明 | 示例 |
|------|------|------|
| 应用软件 | 运行在操作系统之上，用户直接使用 | 微信、Photoshop、财务系统、APP |
| 操作系统 | 系统级软件 | Windows、Linux、Android、iOS |
| 嵌入式软件 | 运行在硬件设备内部（芯片、传感器、控制器） | — |
| 中间件 | 介于操作系统和应用之间 | 数据库中间件、Kafka、Tomcat |

### 软件说明

- **原创**：独立开发的软件
- **修改（含翻译软件、合成软件）**：对原有软件修改后形成的在功能或性能方面有重要改进的软件

### 开发方式

| 方式 | 说明 | 额外材料 |
|------|------|----------|
| 独立开发 | 只依靠自身条件单独开发完成 | 无需 |
| 合作开发 | 两个以上的自然人、法人或非法人组织合作开发 | 书面合作开发合同或协议 |
| 委托开发 | 接受自然人、法人或非法人组织委托开发 | 书面委托开发合同或协议 |
| 下达任务开发 | 由国家机关下达任务开发 | 书面项目任务书或合同 |

**注意**：合作开发协议的签订日期要在开发完成日期前 **3个月以上**，建议 **6个月以前**。

### 发表状态

- **未发表**：软件还未上线，自己还在开发过程中，或仅自己申请
- **已发表**：已经公之于众

## 第4页：软件功能与特点

根据项目具体开发情况撰写，需要贴合操作手册和代码内容。此部分内容较多。

## 第5页：鉴别材料上传

### 材料要求

需上传 **2份PDF格式** 文件：

| 材料 | 要求 |
|------|------|
| 源代码文档 | 前30页 + 后30页，每页不少于50行 |
| 说明书文档 | 操作手册/用户说明书/设计说明书三选一 |

### 一般交存

- 程序和文档的鉴别材料由源程序和任何一种文档 **前、后各连续30页** 组成
- 整个程序和文档 **不到60页的**，应当提交整个源程序和文档
- 程序每页不少于 **50行**，文档每页不少于 **30行**
- **最后1页结尾一定要是完成的功能模块**
- 操作手册页数无强制要求，一般以 **15页以上** 为佳

### 例外交存

按照《计算机软件著作权登记办法》第十二条规定的方式之一提交。例如：材料中涉及商标信息，可提交商标文件或软著名称说明，避免在审查中被要求补正。

## 第6页：签章页

- 打印签章页
- **手写签名**
- 扫描为PDF文件上传
- **不可以使用电子签名或电子章**

## 第7页：提交与查询

### 审查流程

登记流程分为三个阶段：**受理 → 审查 → 制发证**

| 阶段 | 时间 |
|------|------|
| 受理 | 提交后1-2周 |
| 审查 | 自受理日起60日内完成 |
| 下证 | 一般2-3个月；一次性通过不补正约2个月 |

### 建议

**提前3个月以上开始申请**，避免补正影响下证时间。

### 查询方式

在用户中心查询软著的申请状态。
