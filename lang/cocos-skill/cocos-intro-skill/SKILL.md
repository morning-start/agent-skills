# Cocos Creator 3.8 引擎介绍与使用指引

## 任务目标
- 本 Skill 用于：帮助开发者快速了解 Cocos Creator 3.8 引擎，掌握入门指引
- 能力包含：引擎概述、核心概念、工作流程、新手入门
- 触发条件：初次使用 Cocos Creator、需要了解引擎架构

## 概述

Cocos Creator 3.8 是一款高效、轻量、免费开源的跨平台 2D&3D 图形引擎，也是一个实时 2D&3D 数字内容创作平台。拥有高性能、低功耗、流式加载、跨平台等优点，可用于创作游戏、车机、XR、元宇宙等领域的项目。

## 核心概念

### 版本说明
- **Cocos Creator 3.x**：发布于 2021 年初，完全摒弃 Cocos2d-x 底层，采用全新高性能跨平台 3D 内核
- **Cocos Creator 2.x**：发布于 2018 年，2023 年停止更新
- **注意**：3.x 与 2.x 在用法和 API 上不完全兼容

### v3.8 新增功能
- 程序化动画
- 高精度文本
- 全新的可定制渲染管线
- 角色控制器

### 编辑器界面
- **层级管理器**：以树状列表展示场景中的所有节点和层级关系
- **资源管理器**：显示项目资源文件夹（assets）中的所有资源
- **场景编辑器**：用于展示和编辑场景中可视内容的工作区域
- **动画编辑器**：用于编辑并存储动画数据
- **属性检查器**：查看并编辑当前选中节点和组件属性
- **项目预览**：在 Web 或原生平台预览游戏运行效果
